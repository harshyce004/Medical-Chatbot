# MedBot

A simple React-based health chatbot that gives basic advice for common symptoms.

# project setup

1. Clone this repo: git clone https://github.com/harshyce004/medbot.git
2. cd medbot
3. Install dependencies: npm install
4. Run app: npm start

# Tech stack
Tech Stack
Frontend - React / React Native 
Backend - FastAPI / Python 
Database - ostgreSQL / Firebase 
Authentication - Firebase Auth / Google Auth 
Deployment - Render / Firebase Hosting 

# Backend setup
pip install -r requirements.txt
uvicorn main:app --reload
